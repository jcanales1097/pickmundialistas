Marcar resultados de picks
El marcador del partido se sincroniza solo, pero si un pick (ej. "México o Empate") acertó o falló depende del mercado — márcalo aquí una vez termine el partido para que las estadísticas del sitio se calculen correctamente.
Cómo publicar los picks
1
Pídele al chat los picks del día. El chat te entregará un archivo picks.json listo para descargar.
2
Sube ese archivo aquí arriba (clic en el recuadro o arrastra el archivo).
3
Revisa la vista previa de los picks cargados.
4
Una vez termine cada partido, marca cada pick como ✓ Acertado o ✗ Fallido en la sección de arriba — esto es lo que alimenta la tasa de acierto y estadísticas del sitio.
5
Descarga el picks.json actualizado y súbelo a tu repositorio de GitHub (en la misma carpeta que index.html) para publicar los cambios. El marcador del partido (no el acierto del pick) sí se sincroniza automáticamente en el calendario.
Formato del archivo
Esta es la estructura que debe tener picks.json
{ "picks": [ { "id": "p001", "partidoId": "m001", "fecha": "2026-06-17", "tipo": "Resultado", "mercado": "Doble Oportunidad", "pick": "Portugal o Empate", "cuota": 1.25, "probabilidad": 82, "confianza": "Alta", "modelo": "v3", "justificacion": "Texto breve del razonamiento", "resultado": null } ], "combinadas": [ { "id": "c001", "fecha": "2026-06-17", "tipo": "Conservadora", "riesgo": "Bajo", "cuotaCombinada": 2.10, "descripcion": "Texto descriptivo", "picks": ["p001", "p002"] } ] }

El campo partidoId debe coincidir con el ID del partido oficial (m001 a m072, fase de grupos). Pide al chat la lista si la necesitas.
